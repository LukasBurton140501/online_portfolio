Modeling

When creating games I always make most of the assets that are needed, to do this I need to use different software to create the assests that will be used.

Autodesk Maya

When creating 3D Models I always use Autodesk Maya as I have the most experience with the software and I prefer how it functions compared to 3DS Max. personally I think Maya is increadible and I can only find fault in its tendencys to crash when creating and editing UV shells.

3DS Max

I have learned how to use 3DS Max however I find Maya to be so much more efficent compared to Max because of this fact I prefer Maya and rarly use this program.

Photoshop

I use photoshop to create the textures that will be placed on my Models however I often have the most issues at this stage due to hidden polygons or the modeling software just not wanting to function with photoshop.
